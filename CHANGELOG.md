# Changelog

All notable changes to this template are documented here.

## [1.0.0] - 2026-06
### Added
- 17-tab marketing reporting template (Excel)
- Executive Summary with month dropdown and dynamic scorecard
- KPI Monthly single source of truth with auto-recomputing summary, month-on-month, and quarterly tabs
- By Product, All Pages, By Channel, Paid Campaigns, SEO Keywords, Page Speed, AI Search, Geography, Risks and Fixes tabs
- Definitions and Sources tabs
- Claude prompt library for filling every tab through the Windsor.ai connector
- Total Users as the single volume metric across all tabs
